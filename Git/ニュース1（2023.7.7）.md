# 新SNS! 「Threads」  
- 2023.7.5（アメリカ時間）に公開された米MetaのSNSアプリ「Threads」が1日で登録者3000万人を超えた
- サービス開始から2時間で200万、4時間で500万、7時間で1000万ユーザーを超えており、勢いは増し続けている
